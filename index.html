IdeaPool is a collaboration website for open and closed source projects. It is meant to serve as a frontend to the student mentorship programme and other projects.

How does project mentorship work?

The usual flow is like this - students fill out a form, we select mentors, we assign students to mentors, and then, well, we forget about everything :)

This time, we decided to make the process a bit more interesting. So, this time -

  • anyone can propose a project (sample project)
  • admins approve interesting and doable projects
  • anyone can volunteer for a project
  • admins interview volunteers and choose project members
  • managers set up goals, provide templates, guidance
  • project finishes up, and everyone learns!

How can I participate? What do you need me to know?

If you're willing to learn, that's more than enough. Apart from that, just register on DC website, and use the same credentials here. They should work.
